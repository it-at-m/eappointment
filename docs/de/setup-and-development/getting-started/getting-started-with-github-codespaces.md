# Erste Schritte mit GitHub Codespaces

Diese Seite beschreibt, wie eappointment in GitHub Codespaces betrieben und validiert wird.

## Dokumentationsseite lokal betreiben

Um die VitePress-Dokumentation aus diesem Repository in einem Codespace oder auf deinem Rechner auszuführen, siehe [Erste Schritte mit der Dokumentation](/de/setup-and-development/getting-started/getting-started-with-docs).

## Zweck

Codespaces eignen sich, wenn du eine reproduzierbare Cloud-Entwicklungsumgebung ohne lokale Unterschiede in Docker-/Podman-Setups benötigst.

# Project Overview

## Context

This repository is used to maintain and publish Munich-specific changes for eAppointment.
It is developed and documented in the open to support municipal service delivery.

## Major Project Streams

- **ZMS**: foundational rollout of the appointment management system.
- **MPDZBS**: modernization of citizen-facing components and Vue frontend architecture.
- **ZMSKVR**: additional business features and user experience improvements.
- **MUXDBS**: maturity model and OZG-aligned capabilities (for example identity integrations).

## Main Functional Capabilities

- appointment booking and process initiation
- import of services and locations from external sources
- multi-level organizational scope management
- opening-hours and closing-day management
- role-based login and administration
- ticket printer and call display integrations
- statistics and reporting
- emergency calls for staff

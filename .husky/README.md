# Git Hooks

Husky-managed hooks for this monorepo: `pre-commit`, `commit-msg`.

**Documentation:** [Git hooks (Husky)](../docs/en/setup-and-development/git-hooks.md) (English) · [Git-Hooks (Husky)](../docs/de/setup-and-development/git-hooks.md) (Deutsch)

**Setup (once per clone, repository root):**

```bash
npm install
```

`prepare` runs automatically and wires Git to `.husky/`.
